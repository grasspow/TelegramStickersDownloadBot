TelegramStickersDownloadBot
===============
一个简单的tgbot，它会返回一个包含图片的压缩包  
把贴纸包分享链接发给它即可  

### Requirements  

* Node.js v16.20.0（作者使用的是这个版本，其余的自测）

### Usage

1. git clone
2. 从 [@BotFather](https://telegram.me/BotFather) 获取token
3. 复制一份config.js.example为config.js
3. 根据需要修改config.js
4. `npm install && npm start`
